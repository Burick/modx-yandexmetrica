Extra: Yandex Metrica Dashboard Widget
--------------------
Version: 1.0.0
Since: August 26th, 2013
Author: Ivan Klimchuk, Alroniks Expert Team <ivan@klimchuk.com>

A custom MODx dashboard and widgets that will show data from Yandex Metrica service.

You need a Yandex account to use this plugin.

--------------------
ATTENTION! This plugin require php version 5.4+
--------------------

--------------------
Features
--------------------

- Check your site’s statistics from within MODx
- Information about your site’s visitors, traffic sources, top content, goals and keywords
- Visitors and goals are displayed in graphs
- Traffic Sources are displayed in a pie chart
